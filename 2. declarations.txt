견고한 기본기 HTML&CSS

단위(Units)

단위의 분류
- 절대 길이 단위 : cm, mm, px 등
- 상대 길이 단위 : em, rem, vw, vh, % 등

절대 길이 단위

px
- 화소를 의미하는 pixel 의 약자로 디바이스 화면에서 이미지를 표현하는 가장 작은 단위
사용자가 선언한 고정된 크기를 그대로 화면에 그리기 때문에 직관적으로 쉽게 사용가능하다
- 대부분의 브라우저에서 폰트의 초기 기본 px의 크기는 16px이다

상대 길이 단위

em : 배수 단위. 요소의 글자 크기를 기준점으로 한다.
요소에 글자 크기가 지정되어 있지 않다면, 폰트사이즈를 상속 받는 요소의 경우 부모요소의 글자 크기를 기준점으로 한다.
상속받지 않는 요소의 경우 자신의 기본 폰트 사이즈를 기준으로한다.

% : 부모요소를 기준으로 하는 백분율 단위(em에서의 값 1은 100%입니다. 즉, 부모의 font-size값을 30px로 지정하고, 
그 아래 자식 div의 값에 0.8em을 지정하면 30 * 0.8 = 24px가 됩니다.)

rem : root em. 최상위 요소(<html>)의 글자 크기를 기준점으로 하는 배수 단위

vw : 화면 넓이를 기준으로 하는 백분율 단위

vh : 화면 높이를 기준으로 하는 백문율 단위

vmin : 화면의 넓이와 높이 중 작은값을 기준으로 하는 백분율 단위

vmax : 화면의 넓이와 높이 중 큰 값을 기준으로 하는 백분율 단위

ex : 현재 폰트의 x-height 값. 즉 현재 폰트의 소문자 x 의 높이 값을 기준으로 한 백분율 단위

**font-size를 em단위로 조절하면 아래 chrome 설정에서 글자 크기를 조정하였을 경우 글자가 커지고, 
절대단위로 조절하면 글자 크기를 조정하여도 조정이 되지 않습니다

overflow
: 요소의 콘텐츠가 너무 커다랄 경우 요소를 어떻게 처리할지 지정한다.
(overflow-x, overflow-y 의 축 별로 값을 설정할 수 있습니다.)

visible : overflow 속성의 기본 값입니다. 콘텐츠를 자르지 않습니다.

hidden : 콘텐츠를 요소의 크기만큼 맞추기 위해 잘라냅니다. 스크롤 바를 제공하지 않는다.
(float나 margin 합침 현상 등에서 자식 요소의 넓이나 높이를 포함시키기 위해서도 많이 사용한다.

scroll : 콘텐츠를 요소의 크기만큼 맞추기 위해 잘라냅니다. 
잘려진 나머지 부분을 확인 할 수 있도록 스크롤 바를 제공합니다.

background-image

background-img : url - url을 이용 하여 img를 쓸 수 있다

background-color : 배경에 색을 준다

background-repeat
- repeat : 기본값 이다. 배경을 반복시켜줌
- no-repeat : 배경을 반복하지 않는다
- repeat-x : 이미지를 x축으로 반복
- repeat-y : 이미지를 y축으로 반복
- round : 이미지가 잘리지않게 이미지 크기에 변화를 줘서 반복한다
- space : 이미지가 잘리지 않게 이미지 사이에 공간을 주어 반복한다


background-size 속성

background-size : cover = 이미지를 넓이와 높이에 맞게 잘라냅니다. 반복하지 않음
background-size : contain = 이미지 넓이나 높이에 맞춰 자신의 이미지를 모두 다 넣고 반복합니다.
background-size : 100% =이미지를 width와 height에 맞게 압축합니다

font

font - family : CSS를 적용 하여 글꼴을 변경 할 수 있다. 이때 글꼴을 여러개가 지정가능한데 여러개를
지정하는 이유는 해당 서체가 없을 경우 그 다음 글꼴이 차선책으로 사용되기 위해서이다.
만약 지정된 글꼴이 사용자의 컴퓨터에 설치되어 있지 않을 경우, 글꼴은 브라우저 기본값으로 표시된다.
initial, inherit 을 사용하여 부모의 상속을 받지 않고 기본값으로 초기화 하거나, 부모의 상속을 받기도한다

font-family에서 말씀드렸듯이 지정 글꼴이 설치되지 않을 시 기본값으로 표기됩니다. 
이 때 web fonts를 사용하면 사용자의 컴퓨터에 설치되어 있지 않은 글꼴을 사용할 수 있습니다

web fonts는 원하는 font가 있는 곳으로 연결하여 사용하는 방식을 채택하고 있습니다. 
유명한 web fonts로는 google font가 있습니다

@font-face
: 브라우저 사용자의 컴퓨터에 특정 폰트가 설치되어 있지 않아도 
@font-face 속성을 통해 폰트를 설치하여 사용하도록 할 수 있습니다.


opacity
opacity는 요소의 투명도를 지정 할 수 있습니다. 이때 투명도가 들어간 요소 안의 내용물도 함께 
투명해집니다. 값은 0.0 과 1 사이의 숫자를 지정할 수 있으며 만약 값이 0.5라면 투명도는 50%라고 할 수 있습니다.
